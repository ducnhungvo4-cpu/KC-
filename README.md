# KC画布 MVP

内部影视 AI 创作画布原型，基于无限画布和节点式工作流构建。

## 当前能力

- 无限画布缩放、平移、拖拽和多选
- 文本、生图、生视频节点
- 节点连线、快捷添加、右键添加
- 图片/视频本地导入、拖入、粘贴
- 图片裁剪，并生成新的生图节点
- Seedream 5.0 / Seedance 1.5 Pro 默认模型配置
- 无 API Key 时的本地模拟生图，便于业务试用交互闭环
- 项目导入导出和本地开发预览

## 本地运行

```bash
npm install
npm run dev
```

默认访问：

```text
http://127.0.0.1:5173/
```

也可以直接运行：

```bash
start-dev.cmd
```

## 构建验证

```bash
npm run build
```
